**Цель работы:** Программная реализация генераторов дискретных и непрерывных случайных величин (СВ), визуализация их распределений и статистическая проверка гипотез о соответствии полученных данных теоретическим законам с помощью $\chi^2$.

---

## Часть 1. Дискретная случайная величина (ДСВ)

### 1.1. Алгоритм реализации
Для моделирования ДСВ в методе `simulate_logic` реализован **метод обратной функции распределения** (интервальный метод):
1. Вычисляется массив кумулятивных вероятностей (префиксные суммы) $F_i = \sum_{j=1}^{i} p_j$ с помощью `np.cumsum`.
2. Генерируется число $r \sim U(0, 1)$.
3. Находится минимальный индекс $i$, для которого $r \le F_i$. Значение $X = i+1$ принимается как результат.

Для удобства тестирования реализована функция `randomize_probs`, которая генерирует случайный вектор вероятностей и нормализует его так, чтобы $\sum p_i = 1$.

### 1.2. Статистический анализ
Проверка адекватности модели выполняется по следующим метрикам:
*   **Математическое ожидание и дисперсия:** вычисляются эмпирические значения и их относительная погрешность относительно теоретических параметров.
*   **Критерий $\chi^2$:** 
    $$\chi^2_{emp} = \sum_{i=1}^{m} \frac{(n_i - N \cdot p_i)^2}{N \cdot p_i}$$
    где $n_i$ — количество попаданий в $i$-й класс. Гипотеза $H_0$ принимается, если $\chi^2_{emp} \le \chi^2_{crit}$ при уровне значимости $\alpha = 0.05$ и числе степеней свободы $k-1=4$.

**Пример результатов моделирования (ДСВ):**

| Объем (N) | Погр. мат. ож. (%) | Погр. дисперсии (%) | $\chi^2$ (эмп) | $\chi^2$ (крит) | Результат ($H_0$) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **10** | 12.4% | 35.1% | 2.15 | 9.49 | **TRUE** |
| **100** | 4.2% | 8.6% | 4.12 | 9.49 | **TRUE** |
| **1000** | 0.8% | 1.2% | 3.56 | 9.49 | **TRUE** |
| **10000** | 0.1% | 0.4% | 7.82 | 9.49 | **TRUE** |

## Часть 2. Нормальная случайная величина (НСВ)

### 2.1. Алгоритм реализации
Для моделирования НСВ с параметрами $N(\mu, \sigma^2)$ в коде применен **алгоритм Бокса-Мюллера**. Он позволяет трансформировать две независимые равномерно распределенные СВ $u_1, u_2 \sim U(0, 1)$ в пару независимых нормальных СВ:
$$z_0 = \sqrt{-2 \ln u_1} \cdot \cos(2\pi u_2)$$
Затем значение масштабируется: $X = \mu + z_0 \cdot \sqrt{Var}$.

### 2.2. Проверка
Так как НСВ непрерывна, для применения критерия Пирсона выборка разбивается на интервалы.
*   **Количество интервалов:** вычисляется по формуле логарифмического роста: $k = \log_2 N + 1$.
*   **Теоретические вероятности:** рассчитываются через разность значений функции распределения (CDF) на границах интервалов: $p_i = \Phi(x_{i+1}) - \Phi(x_i)$, используя `scipy.stats.norm.cdf`.
*   **Погрешность:** При $\mu=0$ вместо относительной погрешности рассчитывается абсолютная, чтобы избежать деления на ноль.

**Пример результатов моделирования (НСВ):**

| Объем (N) | Ср. значение (эмп) | Погр. дисперсии (%) | $\chi^2$ (эмп) | $\chi^2$ (крит) | Результат ($H_0$) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **10** | 0.142 (абс) | 42.1% | 1.22 | 5.99 | **TRUE** |
| **100** | 0.051 (абс) | 12.4% | 5.48 | 12.59 | **TRUE** |
| **1000** | 0.004 (абс) | 3.2% | 8.15 | 16.92 | **TRUE** |
| **10000** | 0.001 (абс) | 0.8% | 14.22 | 22.36 | **TRUE** |
