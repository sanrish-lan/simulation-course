
# Лабораторная работа №4. Исследование алгоритмов генерации псевдослучайных чисел

## 1. Постановка задачи и теоретическая база
Целью работы является реализация базового генератора псевдослучайных чисел (ПСЧ), формирование выборки заданного объема и статистический анализ полученной последовательности в сравнении со встроенным генератором языка программирования.

### Свойства идеального датчика
Идеальный базовый датчик должен формировать непрерывную случайную величину $X$, имеющую равномерное распределение на интервале $[0, 1)$. 

Теоретические вероятностные характеристики такого распределения вычисляются аналитически и являются критерием проверки качества алгоритма:
*   **Математическое ожидание:** 
    $$ M[X] = \int_{0}^{1} x \, dx = \frac{x^2}{2} \bigg|_{0}^{1} = \mathbf{0.5} $$
*   **Теоретическая дисперсия:**
    $$ D[X] = \int_{0}^{1} \left(x - \frac{1}{2}\right)^2 dx = \frac{1}{12} \approx \mathbf{0.0833333333} $$

Чем меньше отклонение выборочных характеристик от этих констант, тем выше статистическое качество сгенерированной выборки.

---

## 2. Описание используемых алгоритмов

### 2.1. Разработанный генератор 
В качестве собственного алгоритма был выбран мультипликативный конгруэнтный генератор. Формирование $i$-го элемента целочисленной последовательности происходит по правилу:

$$ x_i = (\beta\cdot x_{i-1})\mod M $$

Для получения дробного значения в диапазоне $[0, 1)$ выполняется нормировка: $u_i = \frac{x_i}{M}$.

**Выбранные системные параметры:**
*   Модуль $M = 2^{63}$ (обеспечивает огромный запас длины периода генерации).
*   Множитель $\beta = 2^{32} + 3 = 4294967299$.
*   Начальное значение (зерно) $x_0 = 42$.

### 2.2. Эталонный генератор 
В качестве эталона применяется функция `random.random()` из стандартной библиотеки Python. Она использует алгоритм «Вихрь Мерсенна» (MT19937)  один из самых надежных современных генераторов ПСЧ, имеющий колоссальный период $2^{19937}-1$.

## 3. Методика оценки

Для сформированных массивов данных вычисляются точечные оценки:
1.  **Выборочное среднее** (оценка $M[X]$): 
    $$ \tilde{m}_x = \frac{1}{N} \sum_{i=1}^{N} u_i $$
2.  **Выборочная дисперсия** (оценка $D[X]$):
    $$ S^2 = \frac{1}{N-1} \sum_{i=1}^{N} (u_i - \tilde{m}_x)^2 $$

Абсолютная ошибка вычислялась как модуль разности между выборочным и теоретическим значениями: $\Delta = |\text{Оценка} - \text{Теория}|$.
